# Quick Start

## Create Your First PWA in 5 Minutes

To create your first PWA application, follow these steps:

**Step 1.** Log into your dashboard and click the **"Create PWA"** button in the left menu.

**Step 2.** Fill in the basic settings on the **"Basic"** tab:
- Enter the app name
- Select the language
- Specify the offer link
- Choose creation method: from Google Play app or custom design
- If you chose Google Play — paste the donor app link

**Step 3.** Go to the **"Domain"** tab and bind your domain:
- Enter the domain name (without https:// and www)
- Set the specified NS servers at your registrar
- Wait for DNS propagation (up to 48 hours) and verify the binding

**Step 4.** Configure content on the **"Content"** tab:
- Edit the app header (name, author, rating, size)
- Upload screenshots
- Fill in the description and tags
- Configure reviews and developer contacts

**Step 5.** On the **"Push"** tab, select pre-created push notifications (optional).

**Step 6.** Configure **"Cloaking"**:
- Select the ad network
- Choose the filtering mode
- Set allowed/blocked geos, OS, and browsers
- Select the white page generation method

**Step 7.** Click **"Complete Creation"** — your PWA is ready to go!
