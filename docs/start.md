# Quick Start

To create your first PWA application, follow these steps:

**Step 1.** Log into your dashboard and click the **"Create PWA"** button in the left menu.

**Step 2.** Fill in the basic settings on the **"Basic"** tab:<br>
- Enter the app name<br>
- Select the app language<br>
- Specify the offer link<br>
- Choose creation method: from Google Play app or custom design (If you chose Google Play — paste the donor app link)

**Step 3.** Go to the **"Domain"** tab and bind your domain:<br>
- Enter the domain name (without https:// and www)<br>
- Set the specified NS servers at your domain registrar<br>
- Wait for DNS propagation and verify the domain binding

!!! note "Note"
    If you chose to create PWA from an app, you can skip **Step 4**.

**Step 4.** Configure content on the **"Content"** tab:<br>
- Edit the app header (name, author, rating, size)<br>
- Upload screenshots<br>
- Fill in the description and tags<br>
- Configure reviews and developer contacts

**Step 5.** On the **"Push"** tab, select pre-created push notifications or proceed to create a push by clicking the **"Create Push"** button (optional).

**Step 6.** Configure **"Cloaking"**:<br>
- Select the ad network<br>
- Choose the filtering mode<br>
- Set allowed/blocked GEOs, OS, and browsers<br>
- Select the safe page generation method

**Step 7.** Click **"Complete Creation"** — your PWA is ready to go!
