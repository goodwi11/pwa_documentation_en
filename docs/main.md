# Home

The **Home** tab displays a table with all created PWA applications.

| Column | Description |
|--------|-------------|
| Name | Your PWA name |
| ID | Unique PWA identifier |
| Domain | Linked domain |
| Language | PWA interface language |
| Created | Creation date |
| Modified | Last modification date |
| Status | Current PWA status |
| Action | Edit and delete buttons |
